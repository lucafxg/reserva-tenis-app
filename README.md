# Tenis App - Club Estudiantes de La Plata

Sistema de reserva de canchas de tenis. App estática (sin backend real), datos en localStorage.

## Credenciales demo

- **Email:** admin@edlp.com  
- **Password:** admin

## Setup local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subí esta carpeta a un repositorio de GitHub
2. En [vercel.com](https://vercel.com), importá el repositorio
3. El framework se detecta automáticamente como **Next.js**
4. Click en **Deploy** — ¡listo!

No necesitás configurar variables de entorno. La app es completamente estática.

## Tecnologías

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui** (componentes incluidos)
- **Framer Motion** (animaciones)
- **Radix UI** (primitivos accesibles)
